# MCP Brasil — Brand Identity

## Conceito Visual
Terminal financeiro profissional. Inspirado em Bloomberg Terminal e dashboards de mercado.
Dark-first. Monospace como voz. Verde como sinal de vida e dados ao vivo.

## Paleta de Cores

- Accent:            #10b981  (verde esmeralda — hover, destaques)

- Danger:            #ef4444  (vermelho — quedas, acima do teto)
- Info / CDI:        #8b5cf6  (violeta — série secundária)
- Info / PIB:        #3b82f6  (azul — série terciária)

## Tipografia
- Títulos display:   Geist / system-ui — peso 700–800, letter-spacing -0.02em
- Subtítulos:        Geist — peso 600, letter-spacing normal
- Corpo:             Geist — peso 400, line-height 1.6
- Labels/tags:       Courier New / monospace — uppercase, letter-spacing 0.12em
- Números:           tabular-nums, font-variant-numeric: tabular-nums

## Princípios de Design
1. Dados primeiro — UI serve os números, não o contrário
2. Minimalismo técnico — sem gradientes decorativos, sem sombras pesadas
3. Verde é vida — só aparece quando há dado ao vivo ou ação disponível
4. Tipografia monospace para labels e tickers (identidade terminal)
5. Animações sutis — fade-in + slide-up, duração ≤ 400ms, ease-out
6. Hover = translate(-2px) + border-color → primary (sem scale)
7. Números grandes com tabular-nums para evitar layout shift

## Tom de Voz
FORMAL TÉCNICO. Sem emojis. Siglas em maiúsculo (SELIC, IPCA, CDI).
Datas no padrão brasileiro. Fontes citadas explicitamente (BCB, CVM, B3, IBGE).
